# portal_web

Flask-приложение главной страницы стека: карточки сервисов, live-статус через Docker API и HTTP probes, диаграмма зависимостей.

## Каталог данных (редактирование)

Список веб-UI, API/внутренних точек и топология графа задаются в **[data/catalog.json](data/catalog.json)** (JSON, версия в поле `version`). При старте приложения файл читается и валидируется (уникальные `id`, рёбра графа только между существующими узлами).

Переопределить путь: переменная окружения **`PORTAL_CATALOG_PATH`** (абсолютный или относительный путь к файлу).

Проверка согласованности имён контейнеров с основным compose:

```bash
python scripts/portal_catalog_lint.py
```

(tребуется `PyYAML`).
